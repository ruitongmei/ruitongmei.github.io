  <div class="pub-title">LLM-collaborative Vietnamese Dependency Parsing Data Augmentation</div>
  <div class="pub-authors"><strong>Ruitong Mei</strong></div>
  <div class="pub-venue"><span class="venue-badge">Invention Patent</span></div>
  <div class="pub-abstract">
    This invention patent proposes a large-language-model collaborative method for augmenting Vietnamese dependency parsing data. Multiple mainstream LLMs are used as annotation models to generate initial dependency trees for Vietnamese sentences. An expert LLM, GPT-4o-mini, scores each round of dependency-tree outputs and provides feedback, driving the annotation models to iteratively improve until a target threshold or maximum number of rounds is reached. For each sentence, the highest-scoring tree among three annotation-model outputs is selected to form high-quality synthetic Vietnamese dependency trees, which are then merged with the original UD Vietnamese training set to train dependency parsing models. The method targets low-resource Vietnamese dependency parsing and aims to improve parsing performance through automatic high-quality data expansion.
